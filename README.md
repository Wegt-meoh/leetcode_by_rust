# leetcode by rust language

## File structure

```txt
|-- src
    |-- input_data   // This is a folder for large input data
    |-- lib.rs
    |-- ...other.rs  // The code files that solve problems
|-- Cargo.lock
|-- Cargo.toml
|-- compose-dev.yaml // For docker dev environment
|-- README.md 
```
